==
@author xuzengqiang  
@date 2018-06-13 15:33:51  
自动生成保存与取消按钮

点击取消时,如果有设置route参数,那么默认跳转到route对应的路由.
```html
<fms-form-footer route="/fms/other-income/index">
</fms-form-footer>
```

你也可以修改保存与确定的按钮名称,为false的时候不显示
```html
<fms-form-footer save-label="确定"
                 cancel-label="关闭"
                 @cancel="cancelHandle">
</fms-form-footer>
```
